Application Overview:

Zen Wellora is a comprehensive personal wellness application designed to help users manage their daily health routines through an intuitive and engaging interface. The app combines habit tracking, mood monitoring, and hydration reminders to promote holistic well-being.

Core Features Implemented :

1. Daily Habit Tracker
2. Mood Journal with Emoji Selector
3. Hydration Reminder System

Advanced Features Implemented :

	Home Screen Widget 
	Today's Progress Overview showing completion rate
	Mood shake sensor implementation.

Key Technical Components :

1. AlarmSoundManager
2. Notification System
3. Data Management
